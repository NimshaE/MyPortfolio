KRAMP

Project Information

  • Category: Web Portal
  • Client: Kramp Group
  • Design Story: Kramp Quiz Site Enhancements
  • Company URL: www.kramp.com

Project Overview

As an Associate Software Engineer specializing in UI/UX, I contributed to the Krump project, a quiz site designed for farmers to win prizes by participating in quizzes. This project had already commenced before I joined the company, and I was tasked with implementing new features and making various design improvements during my initial months.

Initial Assignment and Requirements

Upon joining the project, my primary assignment was to add a Video Intro page. This page was to feature an instructing video explaining the quiz process to the users. Additionally, there were several minor UI design updates required to enhance the overall user experience.

Key Tasks and Enhancements

Video Intro Page:

  • Objective: Create a new page featuring an instructing video.
  • Design: Ensure the page aligns with the existing design aesthetics of the site.
  • Features: Implemented a ReCAPTCHA to prevent underage users from accessing the video.

Welcome Page:

  • Objective: Simplify the user journey by removing prize information.
  • Design: Cleaned up the interface to focus on the quiz introduction and participation instructions.

Loading Countdown Page:

  • Objective: Enhance the user experience during quiz loading.
  • Design: Created a visually engaging countdown page to keep users informed about the quiz start.

User Info Page:

  • Objective: Improve social sharing capabilities.
  • Design: Enhanced the interface for easier sharing of quiz results on social media platforms.
  • Additional Feature: Added an email validation prompt to improve data accuracy and user communication.

Flight 2 Updates:

  • Objective: Implement necessary UI changes as per the staging version updates.
  • Design: Reviewed and updated the design to match the latest changes in the staging environment.

Leaderboard Page:

  • Objective: Encourage user engagement through social sharing.
  • Design: Added a Share button that appears as a popup, making it easier for users to share their achievements.

Result Page:

  • Objective: Provide users with their quiz results in an appealing manner.
  • Design: Enhanced the result page to display scores and achievements more effectively.

Wrap Page:

  • Objective: Conclude the quiz experience with a cohesive final page.
  • Design: Designed a wrap-up page that summarizes the user’s participation and encourages further engagement.

Design Process and Collaboration

Throughout the project, I maintained close communication with the development team to ensure that the designs were implemented accurately. Regular reviews and feedback sessions were held to refine the design updates and ensure alignment with the project’s goals.

Working on the Krump project during my initial months as an Associate Software Engineer-UI/UX provided me with valuable experience in enhancing user interfaces and implementing new features. The project allowed me to contribute significantly to the overall user experience, ensuring that the quiz site was not only functional but also engaging and user-friendly. This experience has been a notable addition to my portfolio, demonstrating my ability to handle ongoing projects and deliver impactful design improvements.
